# 미구현 기능 목록

## 🔴 완전히 미구현된 기능

### 1. 미세먼지 기반 테마 색상 조정
- **위치**: `src/hooks/useClimateTheme.ts`
- **상태**: 함수가 주석 처리됨
- **설명**: `adjustColorForAirQuality` 함수가 주석 처리되어 있어 미세먼지 수치에 따른 동적 테마 색상 조정이 작동하지 않음
- **코드 위치**: 86-120번째 줄

### 2. 실제 고도 데이터 연동
- **위치**: `src/services/launchAnalysisApi.ts`
- **상태**: 기본값(50m)만 사용
- **설명**: 발사 환경 분석에서 고도 데이터가 실제 API에서 가져오지 않고 하드코딩된 기본값만 사용
- **코드 위치**: 417-418번째 줄
- **주석**: "고도 (기본값, 실제로는 고도 데이터가 필요)"

### 3. 시간대별 변동 시뮬레이션
- **위치**: `src/services/launchAnalysisApi.ts`
- **상태**: 주석 처리됨
- **설명**: 시간대별 기상 조건 변동 시뮬레이션이 주석 처리되어 있음
- **코드 위치**: 727번째 줄 근처
- **주석**: "시간대별 변동 시뮬레이션 (실제로는 예보 데이터 필요)"

### 5. 고급 필터링

## 🟡 부분적으로만 구현된 기능

### 1. 대기질 데이터 추출
- **위치**: `src/services/weatherApi.ts`
- **상태**: 여러 속성명을 시도하지만 실제 API 응답 구조에 맞게 조정 필요
- **코드 위치**: 463-469번째 줄
- **주석**: "대기질 데이터 추출 (실제 속성명에 맞게 조정 필요)"
- **문제**: 여러 후보 속성명을 시도하지만 실제 API 응답 구조가 확정되지 않음

### 2. 히트맵 시각화
- **위치**: `src/components/HeatmapVisualization.tsx`
- **상태**: 유틸리티 함수만 있고 실제 시각화는 `ClimateMap`에 구현됨
- **설명**: 별도 컴포넌트로 분리되어 있지만 실제로는 `ClimateMap`의 `CircleMarker`로 구현됨
- **주석**: "실제 히트맵 시각화는 ClimateMap 컴포넌트의 CircleMarker로 구현됨"

### 3. 카카오맵 컴포넌트
- **위치**: `src/components/KakaoClimateMap.tsx`
- **상태**: 레거시 파일로 남아있음
- **설명**: 현재는 Leaflet 기반 `ClimateMap`을 사용하지만 카카오맵 컴포넌트가 아직 삭제되지 않음
- **권장**: 사용하지 않는다면 삭제 권장
